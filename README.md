# Tippon

Judo tournament prediction app — run locally or deploy (e.g. Vercel).

## Run locally

**Prerequisites:** Node.js

1. Install dependencies: `npm install`
2. Set `GEMINI_API_KEY` and Supabase vars in `.env.local` (see `CLAUDE.md`)
3. Run: `npm run dev` (port 3000)
